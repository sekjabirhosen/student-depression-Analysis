# Student Depression Analysis  
This project analyzes mental health trends among students using Python.  

## 📂 Dataset  
- `student_depression_dataset.csv` contains demographic, academic, and mental health factors.  

## 📊 Steps in Analysis  
1. Data Cleaning & Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Visualizations (Seaborn, Matplotlib)  
4. Key Findings & Insights  

## 🛠️ How to Run  
- Install dependencies: `pip install pandas numpy seaborn matplotlib`  
- Run `Student_Depression_Analysis.ipynb` in Jupyter Notebook  

## 📈 Key Insights  
- High academic pressure increases depression risk.  
- Poor sleep habits correlate with higher stress levels.
